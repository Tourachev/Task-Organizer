# Task Organizer

## Summary

An online based task organizer intended to optiize workflow, set reminders and take notes. Primarily completed as a training project.

## Stack

-   React
-   Express
-   MongoDb
-   NodeJS

## Functionality

### Notes

-   Create Notes

### Calendar

-   Attach notes and reminders to days

### Reminders

-   Create Reminders
